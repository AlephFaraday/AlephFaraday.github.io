  • 目前主要爭議

    • 捷運局簡報現處理狀況：待補
    • 反對意見
      1. 道路路寬縮減、消防安全疑慮
      2. 停車使用習慣改變（停在自家樓下或橫插）

    • 高捷團隊應對（對應不同路段）

      1. 北移案（美術館）
      2. 路樹就地保護
      3. 機車量

    • 公民團體表達立場

      • A. 台灣城市論壇 徐瑋澤醫生

        • 依層次從三個階段討論支持建設的理由：價值》系統〉建設——

          1. 高齡比（2050+）與移動受限人口
          2. 都市移入人口
          3. 國內主要能源種類
          4. 未來理想的居住環境
          5. 醫學證據支持：多綠地、封街（都市計畫 x 公共衛生）有助於市民降低心血管疾病

          []Compact cities model 緊湊都市模型

        • 呼籲與結論：

          • 回應「提議地下化鐵路」的群眾：應該視輕軌為末端接駁的大眾運輸，而非高高運量運輸（地下鐵路）。
          • 回應「路線發展以輻射還是環線為主？」問題：國外大型都市多先延主要幹道發展輻射路線，再是環線。原因是較不容易發生幹道的交叉。如果高雄選用環狀線，規模應該繼續發展，讓載量上來後才有效益。

      • B. 社團法人 高雄市高雄好過日 陳虹因 協會專員

        • 對於停建的質疑——
        1. 多數民意？
        2. 改路線或立體化？
        3. 大順路的交通惡化的急迫性？四個即將建成的百貨
        4. 拼經濟與輕軌的不容性？樂齡觀光？
        • 建議在一階輕軌的基礎上改善——
        1. 班距過長？
        2. 號誌不清？
        3. 舊編制人力不足，故成立大眾運輸委員會？
