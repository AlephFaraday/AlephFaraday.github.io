Rules．φαινόμενον & νούμενον

Posted on 2019-01-10

規則
標題：「 φ/v + 來源分類・追蹤議題・文章名稱 」
頻率：每週至少一次。
附圖：每篇少於兩張。
其他：
下一百字以內的摘要。
使用德文作為第三語言。

Rules

How to set a title: “?/v + source class?tracking issue?article title”
Post once a week at least.
Use less than 2 pictures for each post.
Others:
Set abstract in 100 words for each article.
Use Deutsch as third language.

φαινόμενον

——此類文章乃透過主體的感知與認知，對於客體的描述與紀錄。來源分類如下：
Phenomenal record - This class of articles refer to phenomena that can be apprehended by human sense a/or perception.
個人觀察筆記 - O
引用的中立或多方平衡報導（須標注來源）- R
第三方資料 - D
未解的猜想 - H

νούμενον

——此類乃不藉由感知與認知理解的存在，所謂真實。
Noumenal fact - This class of articles refer to noumena that exist independently of human sense a/or perception.
